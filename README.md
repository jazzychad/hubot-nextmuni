#hubot-nextmuni

A hubot script for getting SF Muni bus predictions

## Installation

Add **hubot-nextmuni** to your `package.json` file:

```json
"dependencies": {
  "hubot": ">= 2.5.1",
  "hubot-scripts": ">= 2.4.2",
  "hubot-nextmuni": ">= 0.0.0"
}
```

Add **hubot-nextmuni** to your `external-scripts.json`:

```json
["hubot-nextmuni"]
```

Run `npm install hubot-nextmuni`
